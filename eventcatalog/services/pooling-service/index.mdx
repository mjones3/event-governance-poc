# Pooling Service

**Domain**: Manufacturing
**Repository**: biopro-manufacturing

## Overview

The pooling service is responsible for handling pooling operations within the BioPro Manufacturing domain.

## Events Published

This service publishes 4 events:

- **PooledProductCompleted**: PooledProductCompletedEvent
- **PooledProductCreated**: PooledProductCreatedEvent
- **PooledProductUpdated**: PooledProductUpdatedEvent
- **UnitUnsuitable**: UnitUnsuitableEvent

## Events Consumed

This service consumes 5 events:

- **ProductUnsuitable**: Consumed from upstream services
- **DeviceCreated**: Consumed from upstream services
- **DeviceUpdated**: Consumed from upstream services
- **UnitUnsuitable**: Consumed from upstream services
- **ProductDiscarded**: Consumed from upstream services

## Technology Stack

- **Framework**: Spring Boot with WebFlux (Reactive)
- **Messaging**: Apache Kafka with Avro serialization
- **Schema Registry**: Confluent Schema Registry
- **Error Handling**: DLQ pattern with AbstractListener base class
