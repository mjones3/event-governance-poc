# Irradiation Service

**Domain**: Distribution
**Repository**: biopro-distribution

## Overview

The irradiation service is responsible for handling irradiation operations within the BioPro Distribution domain.

## Events Published

This service publishes 2 events:

- **ProductModified**: ProductModifiedEvent
- **ProductQuarantined**: ProductQuarantinedEvent

## Events Consumed

This service consumes 6 events:

- **DeviceCreated**: Consumed from upstream services
- **ProductStored**: Consumed from upstream services

- **ProductModifiedKafka**: Consumed from upstream services
- **ProductQuarantinedKafka**: Consumed from upstream services

- **Framework**: Spring Boot with WebFlux (Reactive)

- **Schema Registry**: Confluent Schema Registry
- **Error Handling**: DLQ pattern with AbstractListener base class
