# Manufacturing Service

BioPro microservice that manages the complete lifecycle of apheresis plasma products from creation through completion.

## Published Events

- **ApheresisPlasmaProductCreated** (v2.3) - When new product is created
- **ApheresisPlasmaProductUpdated** (v1.0) - When product information changes
- **ApheresisPlasmaProductCompleted** (v1.0) - When manufacturing is completed
- **ApheresisPlasmaProductUnsuitable** (v1) - When product fails quality checks

## Domain

**Manufacturing**

## Infrastructure

- **Kafka Broker**: Redpanda
- **Schema Registry**: Confluent Schema Registry
- **Serialization**: Avro with schema validation
- **DLQ**: Dead Letter Queue for failed messages

## Metrics

- Schema validation success/failure rates
- Event publish success/failure rates
- DLQ routing metrics
